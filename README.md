# Bookshelf v2
PRはv2ブランチに投げてください.

### セットアップ

```bash
$ make setup
```

### 各APIキーの設定

.envの `FACEBOOK_ID`, `FACEBOOK_SECRET`, `TWITTER_ID`, `TWITTER_SECRET`, `GITHUB_ID`, `GITHUB_SECRET` に各自で取得した値を設定してください.

### ローカルサーバの起動

```bash
$ php artisan serve
```

### Sassのコンパイル

```bash
$ npm run watch
```
